## 1️⃣ Bug 修复

### 问题
```
配置编辑器错误: name 'QComboBox' is not defined
```

### 原因
`QComboBox` 在 `ConfigFileEditor` 类中使用，但未在导入语句中声明。

### 解决方案
在 `meow_parser.py` 第 17-23 行，添加 `QComboBox` 到导入列表：

```python
from PyQt6.QtWidgets import (
    QApplication, QSystemTrayIcon, QMenu, QWidget, QVBoxLayout, 
    QHBoxLayout, QLineEdit, QPushButton, QTextEdit, QLabel,
    QTreeWidget, QTreeWidgetItem, QDialog, QCheckBox, QMessageBox,
    QFrame, QComboBox  # ← 新增
)
```

### 验证
- ✅ 代码语法检查通过
- ✅ 无诊断错误
- ✅ 用户确认可运行（Python 3.11）
