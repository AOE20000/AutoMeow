# MeowParser - 喵语解析器

智能文本处理工具，通过输入监听和悬浮窗实现实时文本转换

基于 PyQt6 构建，支持跨平台运行（Windows/Linux/macOS）

**当前版本：2.2.0**

## 安装依赖

```bash
pip install -r requirements_pyqt6.txt
```

## 功能特点

- 系统托盘运行，占用资源少
- 红/绿图标显示启用状态
- 智能输入监听，自动弹出悬浮窗
- 悬浮输入窗口，不干扰原有输入框
- 多配置文件系统，快速切换不同场景
- 智能文本替换功能（支持全文匹配和正则表达式）
- 规则组管理，灵活组织替换规则
- 单实例运行保护
- 窗口白名单管理
- 配置导入/导出，方便分享和备份

## 工作原理

启用后，当你在白名单窗口中开始输入时：
1. 程序监听你的键盘输入
2. 当你激活输入框（点击输入框或输入任意内容）后，按空格键时自动弹出悬浮窗
3. 悬浮窗会强制获取焦点，阻止输入进入原输入框
4. 原输入框的内容保持不变，不会被删除
5. 在悬浮窗中输入你要发送的文本
6. 按回车：应用替换规则，然后自动输入到原位置并回车
7. 按Ctrl+回车：直接发送原始内容，不做任何替换
8. 按ESC：取消输入，关闭悬浮窗

注意：悬浮窗仅在"激活输入框"后按空格键时触发，确保不会误触发。悬浮窗会强制劫持焦点，确保输入只进入悬浮窗。

## 使用方法

1. 运行程序后会在系统托盘显示一个"喵"字图标

> **计划进行美化**

2. 图标颜色表示状态：
   - 红色：禁用状态
   - 绿色：启用状态
3. 右键点击图标可以：
   - 启用/禁用功能
   - 打开白名单窗口管理
   - 打开配置文件编辑窗口
   - 查看权限状态
   - 打开调试窗口
   - 退出程序
4. 在窗口管理中：
   - 可以搜索窗口
   - 双击窗口切换启用状态
   - 刷新窗口列表
5. 在配置文件管理中：
   - 可以创建多个配置文件，分类管理不同场景
   - 通过下拉菜单快速切换配置
   - 规则组用于UI组织，规则按文件顺序执行
   - 可以添加、编辑、删除规则组和规则
   - 支持导入/导出配置文件，方便分享和备份
   - 配置文件存储在 `.kiro/rules/` 目录
6. 权限状态显示：
   - ✓ 管理员：可以正常使用所有功能
   - ✗ 普通用户：功能受限，需要以管理员身份运行

## 快捷键

- **回车键**：触发文本替换并自动发送
- **Ctrl + 回车**：发送原始内容，不进行任何替换
- **ESC键**：取消输入，关闭悬浮窗
- **Ctrl + Shift + Alt + M**：快速切换当前窗口的启用状态

## 调试功能

程序内置了调试窗口，方便测试和排查问题：

1. 右键托盘图标选择"调试窗口"打开
2. 调试窗口包含两个部分：
   - **日志输出**：显示程序运行的详细日志
   - **测试输入框**：可直接测试替换功能
3. 日志功能：
   - 查看文本处理过程
   - 查看焦点恢复状态
   - 查看错误信息
   - 可清空或复制日志
4. 调试窗口会在打开窗口管理后自动锁定到白名单

## 文本替换功能

所有文本处理逻辑都通过配置文件管理，包括"喵"字添加功能。

### 配置文件系统（v2.2.0）
- 支持多个配置文件，每个配置文件是一个完整的规则集
- 通过下拉菜单快速切换不同场景的配置
- 配置文件存储在 `.kiro/rules/` 目录
- 支持配置导入/导出，方便分享和备份
- 自动迁移旧版本配置到新格式

### 规则组管理
- 规则组仅用于UI组织和折叠，方便查看
- 规则按配置文件中的顺序依次执行，确定性强
- 每个规则组可以包含多条替换规则
- 支持规则组重命名和删除
- 所有文本处理都通过规则实现，没有硬编码逻辑

### 替换规则
当按下回车键时：
- 程序会按配置文件中的顺序依次应用所有规则
- 支持普通文本匹配和正则表达式匹配两种模式
- 每个规则会替换文本中所有匹配的位置
- 规则执行顺序：按配置文件中从上到下的顺序

### 默认配置

程序默认包含"喵语转换"配置文件：

**喵语转换规则组**
- 在标点符号前添加"喵"
- 在换行符前添加"喵"
- 在句尾添加"喵"

你可以创建自己的配置文件，或导入其他人分享的配置。

### 使用示例

普通文本示例：
- 原句："为什么？"
- 应用"喵语转换"配置
- 结果："为什么喵？"

多规则示例：
- 创建配置"人称替换"，包含规则："我" → "喵"
- 添加规则组"喵语转换"
- 原句："我不知道"
- 结果："喵不知道喵"

正则表达式示例：
- 创建规则："\\d+" → "很多"（正则）
- 添加喵语转换规则
- 原句："我有123个苹果"
- 结果："我有很多个苹果喵"

配置切换示例：
- 配置A："工作场景"（正式用语）
- 配置B："朋友聊天"（轻松用语）
- 通过下拉菜单快速切换，无需重启程序

## 注意事项

- 需要管理员权限运行（因为需要键盘监听）
- 启用状态下会监听键盘输入
- 建议在聊天软件等适当场合使用
- 悬浮窗会自动置顶，输入完成后自动隐藏
- 程序会自动记录鼠标位置，用于点击目标输入框
